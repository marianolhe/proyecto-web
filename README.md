# 📚 Mar's Book Blog App (CRUD)

## 📌 Descripción

Aplicación web tipo blog de libros que permite realizar operaciones CRUD (Create, Read, Update, Delete) consumiendo una API REST.

El usuario puede:

* Ver listado de libros
* Ver detalle de un libro
* Crear nuevos libros
* Editar libros existentes
* Eliminar libros
* Agregar libros a favoritos

---

## 🌐 API utilizada

MockAPI:

https://69daf8b2560857310a073292.mockapi.io/proyecto/books

Campos del recurso:

* id
* title
* author
* genre
* year
* description
* cover
* rating
* createdAt
